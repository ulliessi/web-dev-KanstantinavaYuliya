Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

What is selector’s specificity and how it's calculated?

Selector's specificity is a mechanism in CSS that determines which styles are applied to an element when there are conflicting style rules. It's calculated based on a set of rules:

  • Inline styles have the highest specificity.
  • Next, ID selectors have a higher specificity than class and attribute selectors.
  • Class selectors, attribute selectors, and pseudo-classes have the same level of specificity.
  • The lowest specificity is for type selectors and pseudo-elements.
The specificity is calculated as a score, and the selector with the highest score will have its styles applied.

What are complex selectors?

Complex selectors are combinations of simple selectors, separated by combinators. Combinators are symbols that define the relationship between the simple selectors. The four types of combinators in CSS are:

  • Descendant combinator (space)
  • Child combinator (>)
  • Adjacent sibling combinator (+)
  • General sibling combinator (~)

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes and pseudo-elements are special keywords in CSS that can be used to style specific parts of an element. The difference between them lies in their usage:

  • Pseudo-classes are used to define a special state of an element. For example, :hover can be used to change a button's color when the user's pointer hovers over it.
  • Pseudo-elements are used to style specific parts of an element. For example, ::first-line can be used to change the font of the first line of a paragraph.
